# simple_timer

仅仅是一个简单的计时器

算是重复造轮子吧

使用:

新建一个计时器:

    timer = simple_timer.Timer()
    
获取计时器的时间:
    
    timer_time = timer.time()
    
开始计时器:

    timer.start()
    
暂停计时器:

    timer.stop()
    
重置计时器:

    timer.reset()
